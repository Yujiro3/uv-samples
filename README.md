# uv-samples
======================
sample collection of libuv

[uvbook](https://github.com/nikhilm/uvbook)


利用方法
------

### libubモジュールのインストール ###
    
    $ sudo aptitude install -y libuv-dev 

### クイックスタート ###
    
    $ cd fcgi-cli
    $ make
    $ ./fcgi-cli
